### 結論

### 具體調整項目

1. **新增 YAML frontmatter**：在檔案開頭加入：
   ```yaml

   name: startup-check
   description: "Session 啟動檢查工具。執行 Handoff 任務恢復、Git 環境檢查、開發狀態確認。Use for: (1) 新 session 開始時環境檢查, (2) 偵測待恢復的 handoff 任務, (3) 確認 Git 和開發環境狀態。"
   ---
   ```
2. **建議設定 `disable-model-invocation: true`**：此 Skill 是 Task 型（有副作用：主動恢復任務），應防止 Claude 擅自執行
