**合規狀態**: 需要重寫

|---|---------|------|------|

### 結論

需要重寫。此為所有 agent 中最長的檔案（2985 行），嚴重違反 Progressive Disclosure 原則。檔案長度為建議值的 6 倍。大量 MCP 工具技術細節不屬於 agent 定義。

### 具體調整項目

1. **frontmatter 修正**：移除 `tools`, `color`, `model`，改用標準欄位
2. **大幅精簡**：目標 < 300 行。主體僅保留觸發條件、核心職責摘要、禁止行為、邊界、升級機制
3. **Progressive Disclosure**：MCP 工具使用指南、Serena 工具指南、Context7 工具指南、詳細工作流程範例全部移至 references/
4. **移除 MCP 技術細節**：API 簽名和參數列表應引用 MCP 官方文件，不在 agent 定義中重複
5. **移除所有 emoji**
6. **移除搜尋工具區段**
