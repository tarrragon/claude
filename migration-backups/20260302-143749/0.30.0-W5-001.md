# 0.30.0-W5-001: 依照 skill-design-guide 重新檢視所有 agents 和 skills

## 目標
skill-design-guide 已更新，需依照新規範重新檢視所有既有的 agents（27 個）和 skills（32 個），評估每個是否符合新規範，並記錄需要重寫或調整的項目。

## 執行策略
1. 為每個 agent 和 skill 建立獨立子任務 Ticket
2. 使用 parallel-evaluation 並行評估所有子任務
3. 評估結果寫入各子任務 Ticket
4. 依評估報告著手改寫

## 子任務清單
- Agents: W5-002 ~ W5-028（27 個）
- Skills: W5-029 ~ W5-060（32 個）
